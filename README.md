# Pysteroids
Pysteroids is a [Pygame](https://www.pygame.org/wiki/about) implementation of the classic game Asteroids.

## How to Run
1. Ensure you have Python 3.13 or later and [uv](https://github.com/astral-sh/uv) installed.
2. Install dependencies: `uv pip install -e .`
3. Run the game: `uv run python main.py`
